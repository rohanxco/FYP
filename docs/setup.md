# Project Setup Guide

## Requirements
- Flutter SDK (latest stable version)
- Android Studio or Xcode
- Git
- Google Maps API key (if using Google Maps)
- Device or emulator with location services enabled

## Installation
1. Clone the repository:
   git clone https://github.com/rohanxco/FYP.git
   cd FYP

2. Install dependencies:
   flutter pub get

3. Run the application:
   flutter run

## Configuration
- Add API keys in `lib/config/api_keys.dart`
- Enable GPS and compass permissions in AndroidManifest.xml and Info.plist

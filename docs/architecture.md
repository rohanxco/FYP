# System Architecture

## Overview
The application follows a layered architecture divided into:
- Presentation Layer
- Domain Layer
- Data Layer
- Infrastructure Layer

## Presentation Layer
Responsible for UI components, widgets and user interaction.

## Domain Layer
Contains business logic:
- Prayer time calculations
- Qibla direction mathematics
- Mosque search logic

## Data Layer
Interfaces with:
- Location services
- Maps API
- Local storage
- Sensors (compass, GPS)

## Infrastructure
- API clients
- Service wrappers
- Notification scheduler

Architecture diagrams will be added as the project evolves.
